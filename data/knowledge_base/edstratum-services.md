# EdStratum Service Areas

EdStratum Labs offers professional services across six core areas. Each area can be engaged independently or combined into a broader implementation engagement. The common thread is production-grade AI systems grounded in real data, not prototypes or slide decks.

## Canvas LMS Integration

EdStratum builds purpose-built plugins, custom API workflows, and LTI integrations that extend Instructure Canvas beyond out-of-the-box limits. This is one of EdStratum's deepest areas of expertise.

### What's Included
- Custom LTI 1.3 tool development with proper launch security, OIDC login, and ID token verification
- Canvas Data 2 pipeline architecture for reporting beyond real-time API calls
- Gradebook automation and reporting workflows
- Single sign-on (SSO) configuration and roster sync via Names and Role Provisioning Services
- Assignment and Grade Services (AGS) integration for grade passback and line-item workflows
- Deep Linking content selection integrations
- Custom Canvas API workflows and REST API integration
- Developer Key configuration and scoped API access

### Good Fit Indicators
A team needs Canvas integration work when they have a workflow that the default LMS configuration cannot support. Common signals: a custom tool that needs LTI launch data, grade passback requirements, roster-aware user experiences, reporting pipelines that need Canvas Data 2, or a bridge between Canvas and another product system.

### Typical Engagement
Canvas work typically begins with identifying the placement, user role, data flow, permission boundary, and evaluation metric. The first deliverable is often a maintainable integration plan before custom tool development begins.

## AI Implementation Strategy

EdStratum creates grounded, evidence-driven roadmaps that identify where AI creates measurable ROI and where it does not. This service is for leadership teams that need a defensible first use case before investing in production engineering.

### What's Included
- LLM selection and evaluation frameworks
- Build-versus-buy analysis
- Total cost of ownership (TCO) models
- Responsible AI policy scaffolding
- Risk review and compliance audit trails
- Data readiness assessment
- Pilot design with measurable stop/go criteria
- Vendor evaluation and RFP support

### Good Fit Indicators
Executives or product leaders who need a practical AI roadmap, vendor review, pilot design, data readiness assessment, governance baseline, or a clear stop/go decision before investing in production engineering.

### Approach
An EdStratum AI roadmap starts with the workflow, not the model. The first pass identifies the user group, the current manual process, the source data, the integration surface, and the measurable outcome. A roadmap is not ready for implementation until the team can state what would count as success and what would count as evidence to stop.

## Production AI Engineering

EdStratum develops AI-powered features from prompt architecture and RAG pipelines to production deployment and observability. This is the implementation layer — where strategy becomes shipped code.

### What's Included
- Vector database setup and retrieval evaluation
- RAG pipeline architecture with hybrid retrieval (BM25 + semantic), reranking, and confidence scoring
- Prompt system design and structured output integration
- API integration and tool-calling workflows
- Cost optimization and model monitoring
- Logging, observability, and maintainable release patterns
- Operational handoff so teams can maintain the system after launch

### Good Fit Indicators
Teams that already know the use case and need a production-grade RAG pipeline, tool-calling workflow, evaluation harness, prompt system, structured output integration, or operational monitoring.

### RAG Engineering Specifics
EdStratum's RAG work uses metadata-driven retrieval, semantic chunking, keyword and vector-style semantic matching, reciprocal rank fusion (RRF), and reranking. Every grounded answer surfaces a source confidence badge. When confidence is low, the system does not fabricate — it says so and offers escalation. This low-confidence behavior is part of the evidence-driven brand promise.

## Learning Analytics and Insights

EdStratum transforms LMS event data into intelligence that surfaces intervention signals early and measures whether instruction changes outcomes.

### What's Included
- xAPI or Caliper learner event pipelines
- Predictive engagement scoring
- Custom dashboards built around instructional decisions
- Content testing frameworks
- Early warning indicator identification

### Good Fit Indicators
A team has LMS activity data but no reliable intervention signal. The goal is to tie analytics to a specific decision rather than build a generic reporting layer.

### What's Included
- Architectural review and second opinions
- Vendor evaluation and RFP support
- Team skill development and mentoring
- Board and executive briefings

- Strategic roadmap reviews

### Good Fit Indicators
A team has internal delivery capacity but needs senior judgment, a second opinion on architecture, or a disciplined path from idea to production decision. This service is most useful when the organization is not ready for a full-time AI hire but needs executive-level AI guidance.
